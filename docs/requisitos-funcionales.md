# Requisitos Funcionales - COOPAC-CM

| Código | Requisito |
|---|---|
| RF-01 | El sistema debe mostrar un sitio público institucional. |
| RF-02 | El sistema debe permitir búsqueda global. |
| RF-03 | El sistema debe mostrar noticias publicadas. |
| RF-04 | El sistema debe permitir simulación de créditos. |
| RF-05 | El sistema debe calcular cuota y tabla de amortización. |
| RF-06 | El sistema debe guardar prospectos de crédito. |
| RF-07 | El sistema debe permitir autenticación de usuarios internos. |
| RF-08 | El sistema debe redirigir al panel según rol. |
| RF-09 | El sistema debe gestionar simulaciones. |
| RF-10 | El sistema debe permitir filtros de simulaciones. |
| RF-11 | El sistema debe exportar simulaciones en CSV/XLS. |
| RF-12 | El sistema debe gestionar anuncios. |
| RF-13 | El sistema debe gestionar noticias. |
| RF-14 | El sistema debe gestionar asesores. |
| RF-15 | El sistema debe gestionar usuarios, roles y permisos. |
| RF-16 | El sistema debe proteger rutas mediante roles y permisos. |
