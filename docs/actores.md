# Actores del Sistema COOPAC-CM

| Actor | Descripción | Acceso |
|---|---|---|
| Visitante | Persona que navega por el sitio público de la cooperativa. | Público |
| Prospecto | Visitante interesado que usa el simulador y deja sus datos. | Público |
| Usuario interno | Persona registrada que puede iniciar sesión. | Privado |
| Asesor | Usuario que revisa simulaciones y realiza seguimiento a prospectos. | Panel de simulaciones |
| Administrador | Usuario con acceso general a módulos administrativos. | Panel completo |
| Responsable de Imagen | Usuario encargado de anuncios y noticias. | Gestión de contenidos |
| Responsable de RRHH | Usuario encargado de registrar y administrar asesores. | Módulo RRHH |
